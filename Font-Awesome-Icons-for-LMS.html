Font Awesome 4

Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS. The Font Awesome 4 icon set is part of the Moodle build (as of 2024)

Access Font Awesome documentation for version 4.7 release.

Examples Using Icons

The <i> element is used to display icons from Font Awesome by combining it with specific CSS classes that reference the desired icon. Although <i> originally represented "italic" text, it’s commonly used for icons due to its semantic flexibility and lightweight HTML structure.

Icon in a Heading

Heading with an Icon Example
HTML: Result:
<h3>
  <i class="fa fa-star" aria-hidden="true"></i> Examples Using Icons
</h3>

Examples Using Icons

Explanation:

  • <i> Tag: The <i> element holds the icon and provides a way to style it with CSS.
  • fa Class: This generic class from Font Awesome initiates the icon library, allowing other icon-specific classes to work.
  • fa-star Class: Specifies the exact icon (in this case, a star). Each Font Awesome icon has a unique class, like fa-heart, fa-music, etc.
  • aria-hidden="true": Hides the icon from screen readers when it’s purely decorative. This improves accessibility, as screen readers won’t attempt to interpret non-essential icons.

Icon in a Button

Button with Icon Example
HTML: Result:
<a href="#" class="btn btn-primary">
  <i class="fa fa-thumbs-up" aria-hidden="true"></i> Like
</a> 		Like

Explanation:

  • Button with Icon: By embedding the <i> element within a button, you add visual context, such as a thumbs-up, which is immediately recognizable as a "Like" action.
  • Positioning: Placing the <i> element before the text aligns the icon to the left of the button label.
  • Bootstrap Integration: The button uses the btn btn-primary classes to apply Bootstrap's primary button styling, with Font Awesome's icon enhancing its visual appeal and interactivity.

Practical Use

Using icons with <i> elements enhances user interfaces by adding easily recognizable symbols, improving clarity, and often making interfaces feel more modern and polished. The combination of Bootstrap and Font Awesome provides a cohesive and visually appealing user experience.

This approach can be applied broadly, from buttons and headers to navigation elements, as it keeps the HTML lightweight and ensures icons are easily customizable via CSS.
